# Integrations

Extend the0 platform capabilities by integrating with AI assistants and external tools.

## AI Assistant Integration

- **[MCP (Model Context Protocol)](./mcp)** - Enable AI assistants like Claude Code to manage your trading bots

## Overview

the0 provides integration points that allow AI assistants to interact with the platform programmatically. This enables powerful workflows where AI can help you:

- Deploy and manage trading bots
- Monitor bot health and performance
- Analyze logs and troubleshoot issues
- Explore available custom bots

## Getting Started

To integrate an AI assistant with the0:

1. Generate an API key from [your account settings](https://the0.dev/settings/api-settings)
2. Configure your AI tool to use the0's MCP server
3. Start interacting with your bots through natural language

See the [MCP documentation](./mcp) for detailed setup instructions.
